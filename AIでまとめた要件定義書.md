自責上限：要件定義書
1. コンセプト
「自責」という名の精神的体力の切り売りを、週40時間の労働時間の枠組みではなく、**「週28時間の精神資源」**として再定義する。これは、人間が人間らしくあるための「錨（いかり）」である。

2. 基本リソース設定
週の総自責可能時間: 28時間（100,800秒）

リセット: 毎週月曜 5:00（一週間の始まりを明確に定義）

心拍数基準: 65 bpm（安静時中央値）

3. 二系統の消費ロジック
このアプリは、意識的な自責と無意識の疲弊を異なるレートで削り取る。

【任意自責（意志による反省）】

トリガー: 画面上のボタンを長押ししている間。

消費レート: 1秒の長押しにつき、1秒消費。

配分枠: 28時間のうち、最初の12時間をこの枠に充てる。消費仕切った場合は心拍数自責から一秒ずつ消費する。

意味: 「今、自分は反省している」と自覚できる自責。自分の意志でコントロール可能な領域。

【心拍数自責（無意識の疲弊）】

トリガー: 心拍数が 85 bpm（基準+20）を超えている状態。

消費レート: 1秒の超過につき、2秒消費。

最大消費枠: 1週間で最大 16時間分（加速消費した場合8時間で消える）。

算出方式: 85 bpmを超えた瞬間から下回った瞬間までの経過時間を2倍にしてカウント。

意味: 怒り、焦り、緊張。本人の意志に関わらず、精神的体力が「漏れ出している」状態。これを2倍速で削ることで、速やかに「限界」へ導く。

4. 表示仕様（Dual-Arc Interface）
円形のウォッチフェイスを「人生の防波堤」に見立てる。

左からの青い弧: 意志による自責（12時間分）が積算される。

右からの赤い弧: 生理反応による自責（16時間分）が積算される。

中央の表示: 残り時間を HH:mm:ss で秒単位のカウントダウン表示。

操作感: 画面下部の巨大なボタン。物理的に「押している」間だけ、自分の自責が数字として減っていく実感を付与する。

5. 限界到達時の挙動
自責時間が 00:00:00 になった瞬間、全ての計測とボタン入力を無効化する。 画面には一言、以下のメッセージを常駐させる。

「自責限界に達しました。猫をなでるとかして過ごしてください。」

6. 技術スタック（Wear OS）
OS: Android API 30+ (Kotlin / Jetpack Compose for Wear OS)

計測: Health Services の Passive Data を使用。バックグラウンドでの心拍数スパイクを監視。

保存: DataStore による低消費電力な状態保持。

モード: Ambient Mode 対応。常に「残り時間」が視界に入ることで、無駄な自責を抑制する。
