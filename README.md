# 自責をリソースとして管理する
俺達に必要なのは際限ない自責に打ち立てる錨

自分を責めるという行為そのものに一週間における上限を設けましょう。

これをスマートウォッチの時間と心拍数で管理します。

精神的体力は、緊張や怒られといった外的要因と、シンプル自分を責めるという内的要因の二つに分かれるので、

・心拍数自責＝心拍数が規定値を20上回る事によって生じる自責(合計自責可能時間の28時間から最大16時間減る。消費速度は2倍)

・任意自責＝トグルボタンを押し込み続ける事によりする自責(12時間、任意自責が底を尽きた場合には心拍数自責を消費して自責する事ができる)

心拍数自責時間+任意自責時間=28時間

次の週の月曜朝になったらまた28時間が回復します。

基本計算式は、

労働上限時間×0.7=自責上限合計時間

労働上限時間×0.3=任意自責時間

自責上限時間-任意自責時間=心拍数自責時間

です。肉体労働者なら40時間とかにしてもいいかもしれませんね。(今回は俺の直感で労働上限の40時間に0.7を乗じた28時間を合計自責可能時間としました。色々カスタムしてみてね♡)

この28時間を余らせる事を目的に自責をしてもらう事で、精神的体力の枯渇を防ぎます。
